# Link Shortener
A project  to short links by using Express.js, TypeScript and MongoDB

[Live Demo](http://s.mfakyol.com)

## Tech
- Exporess.js
- TypeScript
- MongoDB
- Mongoose
- ejs

## Features
- SSR
-Multi Language
